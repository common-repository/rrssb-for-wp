Ridiculously Responsive Social Sharing Buttons for WordPress

== Description ==
Ridiculously Responsive Social Sharing Buttons for WordPress. Based on http://kurtnoble.com/labs/rrssb

Available buttons:

* Email
* Facebook
* Tumblr
* LinkedIn
* Twitter
* Reddit
* VK
* Hacker News
* Google+
* Pocket
* Pinterest
* Print
* WhatsApp

The buttons can be displayed before and after the content, enabled/disabled by post type and exclude certain IDs.


== Installation ==
1. Upload the plugin files to the \'/wp-content/plugins/rrssb-for-wp\' directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the \'Plugins\' screen in WordPress
1. Use the Settings->RRSSB for WP screen to configure the plugin

== Frequently Asked Questions ==
= Can I add additional html? =
Yes, the plugin has two filters that allow you to modify the output (rrssb_for_wp_top_html and rrssb_for_wp_bottom_html)
